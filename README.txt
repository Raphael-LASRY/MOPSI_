Avant de lancer le fichier Site_Mopsi.py

- Ouvrir le panneau de configuration
- Aller dans Systeme et Sécurité
- Aller dans Système
- Aller dans Paramètre Système Avancés (à gauche)
- Variables d'environnement
- Variables Système (en bas), trouver la ligne "Path" cliquer sur la ligne
- Modifier
- Nouveau
- Copier coller le chemin d'acces où est le dossier "glpk-4.65"
- OK

Before launching the file Site_Mopsi.py

- Open the configuration pannel
- Go in System & Security
- Go in System
- Environment Variable
- System Variable (bottom), find "Path" ligne and click on it
- Modify
- New
- Copy and Paste the directory path where the folder "glpk4.65" is
- OK
